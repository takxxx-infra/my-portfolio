## 学習目的
- L7 攻撃対策を Terraform でテンプレ化し、複数案件へ再利用できる形に落とし込む。
- ログ可視化基盤を整備し、攻撃検知のリードタイムを短縮する。

## 学んだこと
- AWS Managed Rules と Bot Control の組み合わせ最適化。
- CloudFront + WAF + S3 オリジン構成でのログを Athena で分析し、QuickSight へ可視化するパイプライン。
- Terraform で WAF 設定をモジュール化する際のベストプラクティス。
